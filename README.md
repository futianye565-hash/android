## 📱 项目简介

**每日灵感** 是一款简单实用的名言收藏 Android 应用。它可以帮助你：

- ✨ 每天获取一句经典名言，给你一点灵感
- ❤️ 收藏你喜欢的名言，随时回顾
- ✏️ 自定义添加你自己的名言
- 📊 统计总数量和收藏数量

无论你是需要灵感的创作者，还是喜欢收集美好句子的文艺青年，这款应用都能给你带来惊喜！

---

## 🎯 功能特点

| 功能 | 说明 |
|------|------|
| 📖 **每日名言** | 随机展示一句经典名言，每次打开都有新发现 |
| ❤️ **收藏功能** | 点击收藏按钮，保存你喜欢的名言 |
| ✏️ **自定义添加** | 输入你自己的名言，打造专属灵感库 |
| 📊 **数据统计** | 实时显示总名言数和收藏数 |
| 🗑️ **删除管理** | 长按即可删除不需要的名言 |
| 📤 **分享功能** | 支持分享名言到微信、朋友圈等 |
